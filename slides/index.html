05 / 11 Privacy Layer

Private Deposits via Poseidon

1
Generate Secret Note
Client generates a random secret and nullifier. Never sent to the server.
2
Poseidon Commitment
commitment = poseidon_hash_span([secret, nullifier]) stored on-chain. Amount stays hidden.
3
Merkle Tree Storage
Commitment is inserted into a Poseidon Merkle tree. Root is updated on-chain with each deposit.
4
ZK-style Withdrawal
User provides a Merkle proof. Nullifier prevents double-spend. No link to original deposit.

// Cairo ‚Äî native Poseidon commitment

fn deposit_private(
  ref self: ContractState,
  commitment: felt252,
  amount: u256
) {
  // Verify commitment not seen before
  assert(!self.commitments.read(commitment), 'duplicate');
  self.commitments.write(commitment, true);

  // Insert into Poseidon Merkle tree
  let root = self._update_merkle_root(commitment);
  self.merkle_root.write(root);

  // Route to best yield strategy
  self.router.deposit_to_strategies(amount);
}
